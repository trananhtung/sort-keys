# Changelog

All notable changes to this project are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to

## [0.1.0]

### Added

- Initial release: `sort_keys`, `sort_keys_with`, and `sort_keys_by` — a faithful port of the
  `sort-keys` npm package, with `deep`, `ignore_keys`, and a custom comparator (default UTF-16
  ordering).
